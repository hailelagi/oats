# oats

An in-process, streaming engine that runs on the BEAM!

## Getting started

Via a livebook:
```elixir
Mix.install([:oats])
```

Via hex.pm in elixir:
```elixir
def deps do
  [
    {:oats, "~> 0.1.0"}
  ]
end
```

You should be able to find/replace `:ets` with `:oats` and it just works.
